AI 에이전트 플랫폼 개발 교육과정

오픈소스 기반 엔터프라이즈 AI 에이전트 시스템 구축 마스터 코스

과정 개요

본 교육과정은 프로세스 GPT라는 오픈소스 AI 에이전트 플랫폼을 활용하여 기업용 AI 에이전트 시스템을 구축하고 운영할 수 있는 전문 개발자를 양성하는 종합 교육 프로그램입니다. BPMN 기반의 비즈니스 프로세스 엔진과 최신 AI 기술을 결합하여 실무에서 바로 적용 가능한 실전 중심의 커리큘럼을 제공합니다.

교육 목표
• 오픈소스 AI 에이전트 플랫폼을 활용한 기업용 솔루션 개발 역량 확보
• BPMN 기반 비즈니스 프로세스 엔진과 AI 에이전트의 통합 구현
• Vector RAG 및 Graph RAG 기반 지능형 검색 시스템 구축
• Mem0 기반 장기메모리 시스템 설계 및 구현으로 개인화된 AI 에이전트 개발
• LangGraph를 활용한 복잡한 에이전트 워크플로우 설계 및 구현
• A2A(Agent-to-Agent) 및 MCP(Model Context Protocol) 프로토콜을 통한 에이전트 간 통신 구현
• 온프레미스 환경에서의 AI 시스템 구축 및 운영 실무 경험

이런 분들을 위한 과정입니다

👨‍💻

중급 이상의 파이썬 개발자

🤖

AI/ML 기술을 활용한 비즈니스 솔루션 개발에 관심이 있는 개발자

👥

기업용 AI 에이전트 시스템 구축을 담당하는 개발팀

🏢

오픈소스 기반 AI 플랫폼 도입을 검토하는 기업의 기술진

주요 커리큘럼

1단계: AI 에이전트 기초 및 LLM 프레임워크 (3주)

📖 주요 학습 내용 및 활동

• LangChain & CrewAI 기반 에이전트 개발
• LangGraph를 활용한 복잡한 워크플로우 구현
• React 프레임워크 기반 에이전트 실행 환경
• Advanced RAG 시스템 구현
• 에이전트 장기메모리 시스템

🛠️ 실습 프로젝트

• LangGraph 기반 복잡한 에이전트 워크플로우 구현
• Vector RAG + Graph RAG 하이브리드 시스템 개발
• Mem0 활용 개인화 챗봇 에이전트 구현
• Neo4j 기반 지식 그래프 QA 시스템

2단계: Advanced BPMN 및 비즈니스 프로세스 엔진 (3주)

📖 주요 학습 내용 및 활동

• BPMN 2.0 기반 프로세스 모델링
• 프로세스 GPT 아키텍처 분석발
• 프로세스 레벨 메모리 관리

🛠️ 실습 프로젝트

• 고객 서비스 프로세스 자동화 시스템
• 문서 승인 워크플로우 구현
• 프로세스 GPT 커스터마이징
• 메모리 기반 적응형 프로세스 구현

3단계: MCP 및 A2A 프로토콜 구현 (3주)

📖 주요 학습 내용 및 활동

• Google A2A SDK 활용
• MCP(Model Context Protocol) 통합
• 외부 도구 인터랙션

🛠️ 실습 프로젝트

• 다중 에이전트 협업 시스템
• 외부 서비스 연동 에이전트
• MCP 기반 도구 체인 구현

4단계: 프로세스 GPT 내부 시스템 구현 (2주)

📖 주요 학습 내용 및 활동

• A2A 서비스 구현
• 서비스 아키텍처 설계

🛠️ 실습 프로젝트

• A2A 서비스 API 구현
• 서비스 모니터링 대시보드
• 성능 최적화 실습

5단계: 외부 서비스 및 프레임워크 통합 (3주)

📖 주요 학습 내용 및 활동

• N8N 워크플로우 자동화 연동
• OpenAI API 및 Search API 통합
• 어댑터 패턴 구현

🛠️ 실습 프로젝트

• 통합 워크플로우 자동화 시스템
• AI 기반 검색 서비스
• 커스텀 어댑터 개발

6단계: Computer Use 에이전트 및 Browser Automation (2주)

📖 주요 학습 내용 및 활동

• Browser Use 에이전트 구현
• Computer Use 패턴

🛠️ 실습 프로젝트

• 웹사이트 자동화 에이전트
• 데이터 수집 및 처리 봇
• GUI 기반 업무 자동화

7단계: 졸업 프로젝트 - 통합 AI 에이전트 시스템 (3주)

📖 프로젝트 목표

전체 교육과정에서 학습한 기술을 종합하여 실제 비즈니스 시나리오에 적용 가능한 완전한 AI 에이전트 시스템을 구축합니다.

🛠️ 프로젝트 요구사항

• BPMN 기반 비즈니스 프로세스 설계
• 다중 에이전트 협업 시스템 구현
• 외부 서비스 및 도구 연동
• A2A/MCP 프로토콜 활용
• 완전한 End-to-End 솔루션 구현

📋 예시 프로젝트

• 고객 지원 자동화 시스템: 티켓 분류, 자동 응답, 에스컬레이션
• 문서 처리 자동화: 계약서 분석, 승인 워크플로우, 아카이빙
• 마케팅 캠페인 관리: 콘텐츠 생성, 배포, 성과 분석

8단계: 온프레미스 환경 구축 및 운영 (2주)

📖 주요 학습 내용 및 활동

• 사설 온프레미스 LLM 환경 구축
• 파운데이션 모델 세팅
• 프로세스 GPT 설치 및 운영

🛠️ 실습 프로젝트

• 완전한 온프레미스 AI 플랫폼 구축
• 고가용성 시스템 설계
• 운영 자동화 스크립트 개발

* 상황에 따라 일부 교육내용은 변동될 수 있습니다.
